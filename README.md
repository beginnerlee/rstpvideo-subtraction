# rstpvideo-subtraction
My subtraction video with opencv of python.
This code fetches the rstp image and stores the original video and the subtraction video as a video file.
When executing the code, the play and pause functions can be performed through spacebar, which can be determined whether it is played and stopped through the red circle (play) and black square (stop) at the top left.

If you press the esc key to end the video, you can save it as a video file.
The video file contains the original image and the image of the result of subtraction.
